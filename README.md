# EnzoKey - Sistema de Licenciamento Online

Este repositório contém o código-fonte e a documentação do sistema EnzoKey.

## Estrutura do Repositório

- `/src`: Código-fonte dos projetos backend e frontend
- `/docs`: Documentação técnica do projeto
- `/tests`: Testes automatizados

## Como contribuir

1. Faça um fork do projeto
2. Crie uma branch com a sua feature: `feature/nome-da-feature`
3. Faça commits claros e pequenos
4. Envie o pull request para revisão
